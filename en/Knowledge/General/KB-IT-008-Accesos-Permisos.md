# KB-IT-008: Requesting system access or permissions

**Category:** Access Management  
**Impact:** Variable  
**Estimated resolution time:** 1–3 business days

---

## Problem description / Request

The user needs access to a corporate system, network folder, application, or resource they currently don't have permissions for.

---

## Access request process

All access to Techint corporate systems follows the **Identity and Access Management (IAM)** process and requires approval from the area manager.

### Step 1 — Determine what access is needed

Clearly identify:
- **System**: SAP, SharePoint, network folder, specific application, etc.
- **Access type**: read-only, read/write, administrator.
- **Business justification**: why the access is needed.

### Step 2 — Request manager approval

**Access CANNOT be granted without direct manager approval.**

The standard process:
1. The employee sends an email to their direct manager requesting the access.
2. The manager forwards the approval to the IT access team: `it-accesos@techint.com`.
3. IT processes the request within **1–3 business days** depending on the system.

### Step 3 — Use the Self-Service Access Portal (for enabled systems)

For the following systems, employees can request access directly:
- **Microsoft 365 Groups / Teams**: request from `https://myaccess.microsoft.com`.
- **SharePoint sites**: request access from within the site ("Request Access" button).
- **Azure AD Applications**: request from `https://myapps.microsoft.com` → "Request access to an application".

The manager will receive the request by email to approve or reject.

## Systems and provisioning times

| System | Estimated provisioning time | Requires approval from |
|---|---|---|
| Microsoft 365 / Azure AD | 4–8 hours | Direct manager |
| SAP ERP | 2–3 business days | Direct manager + Finance/Control |
| SAP SuccessFactors (HR) | 1–2 business days | HR |
| SharePoint internal sites | 1–4 hours | Site owner |
| Network folders (file shares) | 1 business day | Direct manager |
| Plant / SCADA systems | 3–5 business days | Manager + Industrial Security |
| Citrix / remote applications | 1–2 business days | Direct manager |

---

## Urgent requests

If access is urgent (blocking critical production or security work):
1. Contact the help desk directly: `+54 11 4018-XXXX` (option 1).
2. State the urgency level and business impact.
3. IT will coordinate an express approval with the area manager.

---

## When to escalate to the help desk

- The access request has been pending for more than 3 business days without a response.
- Urgent access blocking critical operations.
- Technical errors when activating an already approved access.

**Suggested priority:** Medium (Normal). High if blocking operations.
