# KB-IT-009: Equipment renewal or replacement request (PC, laptop)

**Category:** Hardware Request  
**Impact:** Variable  
**Estimated resolution time:** 5–15 business days

---

## Request description

The user needs a new device, wants to renew their current equipment, or is requesting additional hardware (monitor, keyboard, mouse, docking station, headset).

---

## Techint equipment renewal policy

### Equipment lifecycle

| Equipment type | Standard renewal cycle |
|---|---|
| Corporate laptop | 4 years |
| Corporate desktop | 5 years |
| Monitor | 6 years |
| Peripherals (mouse, keyboard) | 3 years or upon failure |
| Headset | 2 years or upon failure |

If the equipment has reached its lifecycle, renewal is automatically approved by IT. If it hasn't yet but there is a technical justification (irreparable failure, critical performance issue), an exception can be requested.

---

## Renewal request process

### Step 1 — Check eligibility

1. Go to the IT Portal: `https://intranet.techint.com/it/mis-equipos`.
2. The portal shows the assignment date for each device and whether it is eligible for renewal.
3. If the device is eligible for renewal → proceed to Step 2.
4. If not yet eligible → complete the exception form (Step 3).

### Step 2 — Complete the standard renewal request

1. Go to `https://intranet.techint.com/it/solicitudes` → **New Request → Hardware → Equipment Renewal**.
2. Fill in:
   - Current device inventory number (label on the device).
   - Type of device requested (laptop, desktop).
   - Justification (lifecycle complete / technical failure).
   - Area and approving manager.
3. Submit request → the direct manager will receive an email for approval.
4. Once approved, IT will coordinate delivery within an estimated **5–15 business days** depending on stock.

### Step 3 — Exception request (device before end of lifecycle)

Requires:
- Detailed description of the technical issue justifying early replacement.
- IT diagnosis confirming the device is not repairable or that repair is not cost-effective.
- Approval from direct manager and the unit's IT manager.

Complete the exception form at: `https://intranet.techint.com/it/solicitudes/excepcion-hardware`.

---

## Additional peripheral requests

For mouse, keyboard, monitor, docking station, headset:
1. Complete the request at `https://intranet.techint.com/it/solicitudes → Hardware → Peripheral`.
2. Requires direct manager approval.
3. Delivery timeframe: **2–5 business days** depending on stock.

---

## When is it an incident vs. a request?

| Situation | Type |
|---|---|
| Device won't turn on / fails and needs urgent replacement | **Incident** (high priority) |
| Requesting a new device due to lifecycle completion | **Request** |
| Requesting an additional device for home office | **Request** |
| Broken mouse or keyboard → request replacement | **Request** (low urgency) |
