# KB-IT-010: IT Onboarding for new employees

**Category:** Onboarding  
**Impact:** High  
**Estimated resolution time:** 1–3 business days of preparation

---

## Description

Guide for the IT team and new employees covering the equipment and access provisioning process on the first day of work at Techint.

---

## Pre-onboarding process (IT)

IT must receive the provisioning request at least **5 business days in advance** before the new employee's start date.

The request must include:
- Full name
- Corporate email assigned by HR
- Area and assigned project
- Country and office location
- Direct manager
- Start date
- Required device profile (see KB-IT-009)

---

## New employee checklist — First day

### Device and basic access

- [ ] Receive the configured corporate device.
- [ ] Log in with the temporary credentials sent to the personal email.
- [ ] Change the temporary password on first login (security requirement).
- [ ] Register MFA authentication methods at `https://aka.ms/MFASetup`:
  - Microsoft Authenticator app (recommended).
  - Mobile phone (SMS or call as backup).

### Register SSPR methods

Go to `https://aka.ms/ssprsetup` to register password recovery methods:
- Personal alternative email (recommended as backup).
- Mobile phone.

### Install the VPN client

1. Download GlobalProtect from: `https://intranet.techint.com/it/vpn-installer`.
2. Install and configure with gateway: `vpn.techint.com`.
3. Log in with corporate credentials.

### Corporate applications

All licensed applications are available at `https://myapps.microsoft.com`:
- Microsoft 365 (Outlook, Teams, SharePoint, OneDrive).
- Vacation management system.
- HR Portal (SAP SuccessFactors) — access activated within the first 48 hours.
- SAP ERP — depending on the position profile.

### Set up email in Outlook

The corporate email is automatically provisioned. Open Outlook → follow the automatic configuration wizard.

## Standard access every employee receives on day one

| System | Access provisioned automatically |
|---|---|
| Microsoft 365 (Exchange, Teams, SharePoint, OneDrive) | ✅ Day 1 |
| Techint Intranet | ✅ Day 1 |
| Vacation management system | ✅ Day 1 |
| SAP SuccessFactors (HR) | ✅ 48 hours |
| VPN GlobalProtect | ✅ Day 1 |
| SAP ERP / operational systems | ⚠️ Role-dependent, requires request |
| Plant system access | ⚠️ Requires safety training |

## If something doesn't work on the first day

1. Contact the direct manager or the IT help desk.
2. The IT help desk gives high priority to new employee incidents.
3. Phone: `+54 11 4018-XXXX` (option 1 → option 3: new employees).
4. Email: `it-newjoiner@techint.com`.

---

## When is it an incident vs. a request?

| Situation | Type |
|---|---|
| Device doesn't arrive on the first day | **Incident** (urgent) |
| Device arrived but is not configured | **Incident** |
| Cannot log in with temporary credentials | **Incident** |
| Needs access to a system beyond the standard set | **Request** |
| Needs additional equipment (second screen, headset) | **Request** |
